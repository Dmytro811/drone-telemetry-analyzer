# Система аналізу телеметрії БПЛА на базі штучного інтелекту

Програмний комплекс для обробки та візуалізації польотних даних форматів ArduPilot (.BIN, .LOG). Проєкт орієнтований на автоматизацію технічного аудиту польотів та розрахунок операційної ефективності.

## Основні модулі та функціонал

### 1. Аналітичне ядро (AnalyticsCore)
* **Фільтрація IMU:** Реалізовано High-pass filter для компенсації гравітаційного зміщення (Gravity Leakage). Це дозволяє уникнути математичного дрейфу при інтегруванні швидкості.
* **Геодезичні розрахунки:** Використання формули Haversine для обчислення реальної дистанції з урахуванням еліпсоїда Землі.
* **Енергоефективність:** Модуль розрахунку витрат пального (ДВЗ) або заряду АКБ (LiPo/Li-ion) залежно від обраного профілю БПЛА.

### 2. Візуалізація (3D Engine)
* **ENU Transformation:** Конвертація глобальних координат WGS84 у локальну метричну систему East-North-Up.
* **Інтерактивна сцена:** Побудова траєкторії з кодуванням кольором за вектором швидкості або часом.

### 3. AI-Інтеграція
* Використання моделі Gemini 1.5 Flash для генерації технічних звітів.
* Автоматичне логування результатів аналізу в `logs/ai_analysis.log` для подальшого аудиту.
